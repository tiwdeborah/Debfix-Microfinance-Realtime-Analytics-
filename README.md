# Debfix Microfinance Realtime Analytics
## Project Overview
This project is a realtime microfinance analytics developed to monitor financial transactions, loan repayment activities, customer behavior, and operational performance for a microfinance institution.
The dashboard was designed to simulate a real-world fintech reporting environment where business stakeholders can track key financial indicators and transaction performance in realtime using interactive business intelligence tools.The solution integrates MySQL, Power BI, Python, and Excel to create a dynamic analytics workflow capable of supporting data-driven decision-making.

## Business Problem
Microfinance institutions process thousands of transactions daily, making it difficult to manually monitor repayment trends, failed transactions, customer activity, and operational performance. This project addresses the challenge by providing:
•	Realtime monitoring of transactions

•	Loan repayment performance tracking

•	Failed and pending transaction analysis

•	Interactive business intelligence reporting

•	Executive-level financial insights

## Objectives
•	To build a realtime analytics dashboard using Power BI DirectQuery.

•	To monitor loan repayment activities dynamically.

•	To analyze successful, pending, and failed transactions.

•	To improve visibility.

•	To create a professional fintech-style reporting solution.
